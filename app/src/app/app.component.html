This website

it is built with Angular and backed by Symfony.

Deployed on a Kubernetes cluster on my desk at home and load balanced between multiple pod instances by Traefik.

This is hosted on a 4x Raspberry PI stack with uninterruptible power supply, smart plug and automated fan control by an ESP8266 via MQTT.

Aside from this website, the Kubernetes cluster also powers a Home Assistant installation with 62 Zigbee2MQTT devices and about 548 entities in my home.

Combined they all are the base for many home automation scripts from light control, door/window contact, rain sensor, bed occupancy sensor, alarm and air purifiers/ quality sensors. And the list keeps growing...
